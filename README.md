# stargate-homelab

## Prerequisites
Things you need before starting:
* `Some spare hardware`
* `Time and will to have fun :)`

## Project structure
```
stargate-homelab
└── automation
    └── ansible    
    └── terraform
└── docker   
└── observability
└── proxmox
```

## Tasks to accomplish
- The idea of this project is to build a homelab using `proxmox` where we can have tools for daily use and to test things for work or just for fun.
